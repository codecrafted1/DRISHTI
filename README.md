Drishti

Drishti is a smart traffic control and incident reporting web platform that helps users report accidents, congestion, and road issues in real time. It enables authorities and citizens to collaboratively improve road safety and traffic management by sharing live updates about traffic conditions and accidents.

✨ Features
📍 Live Incident Reporting
Users can report accidents, breakdowns, and road hazards with location details.
🚗 Traffic Status Updates
View real-time traffic congestion levels in different areas.
🗺️ Interactive Map View
Visualize incidents and traffic flow on an interactive map.
🚨 Emergency Alerts
Highlight critical accident zones for quick awareness.
👥 Community Driven Data
Users contribute to keeping traffic information up-to-date.
🔎 Search & Filter
Easily find incidents by location, type, or severity.

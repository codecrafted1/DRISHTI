# 🚦 Drishti – Traffic Control & Incident Reporting System

Drishti is a web-based traffic monitoring and incident reporting platform that helps users report accidents, congestion, and road issues in real time. It aims to improve road safety and traffic management by enabling quick communication between citizens and authorities.

---

## 🌟 Features

- 📍 Report accidents and traffic incidents with location  
- 🚗 View real-time traffic congestion updates  
- 🗺️ Map-based visualization of incidents  
- 🚨 Highlight critical accident zones  
- 👥 Community-driven reporting system  
- 🔎 Search and filter incidents by location or type  

---

## 🛠️ Tech Stack

- Frontend: React / Next.js  
- Styling: Tailwind CSS / CSS Modules  
- Backend: Node.js / Express (if applicable)  
- Database: MongoDB / Firebase  
- Maps Integration: Google Maps / Mapbox
